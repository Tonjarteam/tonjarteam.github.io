实例：最大子列和

问题 : 最大子列和
给定整数，求的最大值(为方便起见，如果所以数均为负数，则最大子列和为0)
样例 : 输入：4 -3 5 -2 -1 2 6 -2
输出：11
(最大子列:4 -3 5 -2 -1 2 6)

穷举法

  1. for(int i=0;i<n;i++){
  2.     for(int j=i;j<n;j++){
  3.         ThisSum = 0;
  4.         for(int k=i;k<=j;k++)//计算i->j子列和
  5.             ThisSum+=A[k];
  6.         if(ThisSum>MaxSum)//与最大的比较
  7.             MaxSum=ThisSum;
  8.     }
  9. }

可以看出第4，5行被执行次数最多，我们先计算这里。

分治方法，递归

分治策略(divide-and-conquer)
把问题大致分为两个部分，递归地求解，然后把子问题合并在一起，并可能做少量附加工作，最后得到整个问题的解。

在本题中，我们把给定序列分为前后两个部分，那么最终的解是max{前半部分的解，后半部分的解，跨越前后部分的解}

  1. int MaxSubSum(A[],left,right){
  2.     if(left==right){//一个元素，基准情况
  3.         if(A[left>0]) return A[left]
  4.         else return 0;
  5.     }
  6.     center = (left+right)/2;
  7.     MaxLeftSum = MaxSubSum(A,left,center);//左子列最大子列和
  8.     MaxRightSum = MaxSubSum(A,center+1,right);//右子列最大子列和
  9.     MaxLeftBorderSum = leftBorderSum = 0;
  10.     for(int i=center;i>=left;i--){
  11.         leftBorderSum+=A[i];
  12.         if(leftBorderSum>MaxLeftBorderSum)
  13.             MaxLeftBorderSum = leftBorderSum;
  14.     }
  15.     MaxRightBorderSum = RightBorderSum = 0;
  16.     for(int i=center+1;i<=right;i++){
  17.         RightBorderSum+=A[i];
  18.         if(RightBorderSum>MaxRightBorderSum)
  19.             MaxRightBorderSum = RightBorderSum;
  20.     }
  21.     return Max3(MaxLeftSum,MaxRightSum, MaxLeftBorderSum+ MaxRightBorderSum);//三种答案中最大的一个作为答案
  22. }

在线算法

联机算法/在线算法
每输入一个数据就能进行即时处理，在任何一个地方中止都能正确地给出当前的解。
  1. ThisSum = MaxSum = 0;
  2. for(int i=0;i<n;i++){
  3.     ThisSum += A[i]//向右累加
  4.     if(ThisSum > MaxSum)
  5.         MaxSum = ThisSum;
  6.     else if(ThisSum < 0)
  7.         ThisSum = 0;
  8. }

联机算法表述了这个算法的联机特性。
这个算法堪称完美，因为它是一个常量空间，线性时间的联机算法。
这个算法的复杂度分析很容易
但是要理解这个算法的正确性不是那么容易了。

我们根据j,i把数据分为三个部分 （1） j （2） i （3）

紫圈必然大于红圈，因为前面加起来为负
这样循环，(2)中的最大值肯定能被找到，反过来推一开始j=0这样在未变j时MaxSum是前面这个子列的最大子列和，换j后总是用(2)的最大值和现有最大值比，所以拿到的也是目前的最大子列和，以此类推，所以这个算法正确。
