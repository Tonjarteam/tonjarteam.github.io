Rank

ckson wants to know his rank in the class. The professor has posted a list of student numbers and marks. Compute Jackson’s rank in class; that is, if he has the top mark(or is tied for the top mark) his rank is 1; if he has the second best mark(or is tied) his rank is 2, and so on.

Input

The input consist of several test cases. Each case begins with the student number of Jackson, an integer between 10000000 and 99999999. Following the student number are several lines, each containing a student number between 10000000 and 99999999 and a mark between 0 and 100. A line with a student number and mark of 0 terminates each test case. There are no more than 1000 students in the class, and each has a unique student number.

Output

For each test case, output a line giving Jackson’s rank in the class.

Sample Input

20070101
20070102 100
20070101 33
20070103 22
20070106 33
0 0

Sample Output

2

code

#include<iostream>
using namespace std;
int main(){
    int score[101];
    
    int id,the_score=0;
    while(cin>>id){
        the_score =0;
        for(int i=0;i<101;i++)
            score[i]=0;
        int cid,cscore;
        while(cin>>cid>>cscore){
            if(cid==0)break;
            score[cscore]++;
            if(cid==id)
                the_score=cscore;
        }
        int count=1;
        for(int i=the_score+1;i<101;i++)
            count+=score[i];
        cout<<count<<endl;        
    }
    
}
