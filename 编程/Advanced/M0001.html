Moving Tables

The famous ACM (Advanced Computer Maker) Company has rented a floor of a building whose shape is in the following figure.
| 1 | 3 | 5 | 7 |……|399|
| |
| 2 | 4 | 6 | 8 |……|400|
The floor has 200 rooms each on the north side and south side along the corridor. Recently the Company made a plan to reform its system. The reform includes moving a lot of tables between rooms. Because the corridor is narrow and all the tables are big, only one table can pass through the corridor. Some plan is needed to make the moving efficient. The manager figured out the following plan: Moving a table from a room to another room can be done within 10 minutes. When moving a table from room i to room j, the part of the corridor between the front of room i and the front of room j is used. So, during each 10 minutes, several moving between two rooms not sharing the same part of the corridor will be done simultaneously. To make it clear the manager illustrated the possible cases and impossible cases of simultaneous moving.

Table moving Reason
Possible 30 to 50 and 60 to 90 no part corridor is shared
Possible 11 to 12 and 14 to 13 no part corridor is shared
Impossible 20 to 40 and 31 to 80 corridor in front of 31 to 40 is shared
Impossible 1 to 4 and 3 to 6 corridor in front of 3 is shared
Impossible 2 to 8 and 7 to 10 corridor in front of 7 is shared

For each room, at most one table will be either moved in or moved out. Now, the manager seeks out a method to minimize the time to move all the tables. Your job is to write a program to solve the manager’s problem.

Input

The input consists of T test cases. The number of test cases ) (T is given in the first line of the input. Each test case begins with a line containing an integer N , 1<=N<=200 , that represents the number of tables to move. Each of the following N lines contains two positive integers s and t, representing that a table is to move from room number s to room number t (each room number appears at most once in the N lines). From the N+3-rd line, the remaining test cases are listed in the same manner as above.

Output

The output should contain the minimum time in minutes to complete the moving, one per line.

Sample Input

3
4
10 20
30 40
50 60
70 80
2
1 3
2 200
3
10 100
20 80
30 50

Sample Output

10
20
30

code

//可以考虑如果是只有一边房间的情况，两边房间与之类似
//注意可以从房间号大的搬到房间号小的
#include<iostream>
using namespace std;
int main(){
    int t;
    int room[201];
    while(cin>>t){
        for(int j=0;j<t;j++){
            int n;
            cin>>n;
            for(int i=0;i<201;i++)
                room[i]=0;
            for(int i=0;i<n;i++){
                int start,end;
                cin>>start>>end;
                if(start>end){
                    int tmp=start;
                    start=end;
                    end=tmp;
                }
                start=(start+1)/2;
                end=(end+1)/2;
                for(int k=start;k<=end;k++)
                    room[k]++;
            }
            int max=0;
            for(int i=0;i<201;i++)
                if(room[i]>max)max=room[i];
            cout<<max*10<<endl;
        }
    }
    
}
