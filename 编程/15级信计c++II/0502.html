半素数

如果一个正整数可以拆分成两个素数的乘积，则该数被称为是一个半素数，如9=3*3以及15=3*5，所以这两个数都是半素数。现给你一些不大于10000的正整数，要你找出其中的那些数是半素数。

Input

第一行有一个整数T，表示要你判断的整数数的个数。
接下来有T行，每行一个正整数，这些数都不大于10000。

Output

对应的正整数如果是半素数则输出yes，否则输出no。

Sample Input

3
9
11
15

Sample Output

yes
no
yes

code

#include < iostream > 
#include < cmath > 
using namespace std;
bool judge_sushu(int n)
{
    int m = sqrt(n) + 1;
    for (int i = 2; i  <  m; i++)
    {
        if (n%i == 0)
            return 0;
    }
    return 1;
}
bool judge(int n)
{
    int m = sqrt(n) + 1;
    for (int i = 2; i  <  m; i++)
    {
        if (n%i == 0)
        {
            return judge_sushu(i) && judge_sushu(n / i);
        }
    }
    return 0;
}
int main()
{
    int T;
    while (cin  >  >  T)
    {
        for (int i = 0; i  <  T; i++)
        {
            int n;
            cin  >  >  n;
            if (judge(n))
                cout  <  <  "yes" <  < endl;
            else
                cout  <  <  "no" <  < endl;
        }
    }
}
