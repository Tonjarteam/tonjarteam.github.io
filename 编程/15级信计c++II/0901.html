括号匹配问题

在某个字符串（长度不超过100）中有左括号、右括号和大小写字母；规定（与常见的算数式子一样）任何一个左括号都从内到外与在它右边且距离最近的右括号匹配。写一个程序来判断输入的字符串中括号是否匹配。

Input

用栈来判别配对。
Input
输入包括多组数据，每组数据一行，包含一个字符串，只包含左右括号和大小写字母，字符串长度不超过100。

Output

对每组输出数据，输出一行，如果括号匹配则输出YES否则输出NO。

Sample Input

)(rttyy())sss)(

Sample Output

NO

code

#include < iostream > 
using namespace std;
bool check(string str){
    char stack[100];
    int p=0;
    for(int i=0;i < str.length();i++){
        if(str[i]=='('){
            stack[p]='(';
            p++;
        }
        else  if(str[i]==')'){
            if(p==0)
                return 0;
            p--;
        }
//写成如此是为了可以拓展成判别多种括号匹配问题
    }
    if(p!=0)
        return 0;
    return 1;
}
