买书

某书店最近在搞促销，推出了新的销售策略： 每买3本书，书店对3本书中最便宜的那本书作免费处理。例如：某顾客挑了三本书，其书价分别为：10,3,2，则按照销售策略，该顾客只需付13元即可。
当然，顾客完全可以买超过3本书，但此时该顾客要付的总价完全取决于他对这些书的分组。例如：某顾客跳了7本书，其书价分别为：10,3,2,4,6,4,9，且该顾客将这些书按(10, 3, 2), (4, 6, 4) 和(9) 来分组，则该顾客需要付费：32元。
显然，上面的这种分组方式不是最好的，还有更好的分组方式，可以使得该顾客可以付出更少的钱来获得这些书。
现在请你为那些顾客计算一下，他们要购买他们选出的书最少需要付多少钱。

Input

本题有多组测试数据。
每组测试数据包含两行。
第一行为一个整数N(1 ≤ N ≤ 100000)，为某顾客欲购买的书的数量。第二行，包含N个整数，为他欲购买的各本书的价格(≤ 100000)。
当N = 0，表示输入结束。

Output

对每组测试数据，请输出一行一个整数，为要购买这些书，所需付出的最少钱的数目。

Sample Input

4
3 2 3 2
6
6 4 5 5 5 5
0

Sample Output

8
21

code

#include<iostream>
#include<queue>
using namespace std;
int main(){
    int n;
    priority_queue<int> q;
    while(cin>>n){
        int tmp;
        int sum=0;
        if(n==0)break;
        for(int i=0;i<n;i++){
            cin>>tmp;
            sum+=tmp;
            q.push(tmp);
        }
        for(int i=0;i<n;i++){
            if(i%3==2){
                sum-=q.top();
            }
            q.pop();
        }
        cout<<sum<<endl;
    }
}
