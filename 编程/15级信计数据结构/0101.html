循环小数

我们知道如何一个真分数都可以展开成一个小于1的小数，且这个小数要么是有限位小数，要么是无限位的循环小数。
现给你一些真分数，请你算出这些小数的循环节的长度。

Input

本题有多组测试数据。
每行有两个整数p和q(0<p<q)。

Output

对每对数p和q，请输出p/q在展开成小数时其循环节的长度。如何p/q是有限位小数则输出0。

Sample Input

1 6
2 4

Sample Output

1
0

code

#include<iostream>
#include<vector>
using namespace std;
int main(){
    int p,q;
    vector<int> y;
    while(cin>>p>>q){
        bool flag=true;
        int l=0;
        y.clear();
        while(flag){
            y.push_back(p);
            l++;
            p*=10;
            p=p%q;
            for(int i=0;i<y.size();i++){
                if(p==y[i]){
                    cout<<l-i<<endl;
                    flag=false;
                    break;
                }
            }
            if(p==0){
                cout<<0<<endl;
                break;
            }
        }
            
    }    
}
