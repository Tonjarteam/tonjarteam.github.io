第K小数

Problem Description
输入N个整数和一个正整数K(K>=1并且 K<=N)，输出这些数从小到大排序后的第K个数(例如，K=1就是最小值)。其中N<=10^7。

Input

本题有多组测试数据。
每组测试数据的第一行包含二个整数N(<=10^7)和K(K>=1并且 K<=N)。
第二行包含N个整数，为给定的数组。
当N=K=0表示输入结束。

Output

对每组测试数据输出一个整数，为给定数组中的第K小数。

Sample Input

5 2
3 5 2 1 4
0 0

Sample Output

2

#include<cstdio>
#include<iostream>
#include<queue>
int num[10000001];
using namespace std;
void qsort(int start, int end,const int& k) {
    if (start >= end )return;
    int s = start, e = end - 1;
    int key = num[s];
    for(;;){
        for(;s<e&&num[e]>key;e--);
        if(s<e)
            num[s]=num[e];
        else break;
        s++;
        for(;s<e&&num[s]<key;s++);
        if(s<e)
            num[e]=num[s];
        else break;
        e--;
    }
    num[e]=key;
    qsort(start, e,k);
    //if(e+1
    qsort(e+1, end,k);
}
